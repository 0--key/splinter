+++++++++++++++++
Mouse interaction
+++++++++++++++++

    **Note:** Mouse interaction currently works only on Chrome driver.



mouseover
---------

.. highlight:: python

::

    browser.find_by_css('h1').mouseo_ver()

mouseout
--------

.. highlight:: python

::

    browser.find_by_css('h1').mouse_out()

double click
------------

.. highlight:: python

::

    browser.find_by_css('h1').double_click()

right click
-----------

.. highlight:: python

    browser.find_by_cs('h1').right_click()
